# Лаунчер «Сессий» — 10 мокапов на выбор (2026-07-04)

Контекст: живой смоук показал, что фраза-конструктор + env-табы «абсолютно неудобны».
Зафиксировано отдельно (применяется ко всем вариантам):
- **«＋ ещё агент здесь» у панели** — быстрый новый агент в ТОЙ ЖЕ папке/среде (путь не вводить).
- **Убрать чипы избранного** из верхней полосы.
- Папка показывается **полным путём**, не «домашняя».

Легенда иконок сред: ◆ Claude · </> opencode · ◎ codex · >_ shell.

---

## M1 — Одна строка (минимум)
```
＋ Новая:  [◆ Claude ▾]  в  [E:\Scripts\Castellyn ▾]  [▶ Запустить]   ⚙
```
Всё лишнее (профиль/аргументы/SSH) — за ⚙. + быстро, − профиль/аргументы спрятаны.

## M2 — Подписанный грид
```
 Среда    [◆ Claude] [</> opencode] [◎ codex] [>_ shell]
 Папка    [E:\Scripts\Castellyn ........................] [📁]
 Профиль  [cc1 ▾]              Аргументы [--effort high ......]
                                              [▶ Запустить]
```
Всё на виду, аккуратно, не «прыгает». − занимает больше высоты.

## M3 — Карточки сред (клик раскрывает форму)
```
 ┌────────┐ ┌──────────┐ ┌────────┐ ┌────────┐
 │   ◆    │ │   </>    │ │   ◎    │ │   >_   │
 │ Claude │ │ opencode │ │ codex  │ │ shell  │
 └────────┘ └──────────┘ └────────┘ └────────┘
 ── Claude ──  Папка [E:\...Castellyn ▾]  Профиль [cc1 ▾]  [▶]
```
Наглядный выбор среды. − два шага (среда → форма).

## M4 — Командная строка (power-режим)
```
 $ [claude --effort high ...............................]  в [E:\...Castellyn 📁]  [▶]
```
Одно поле «как в терминале» + папка. + мощно/быстро для опытных, − новичку непонятно.

## M5 — Визард (по шагу за раз, крупно)
```
 Шаг 1/3 — среда:
   [ ◆ Claude ]  [ </> opencode ]  [ ◎ codex ]  [ >_ shell ]
                                                   [ Далее → ]
```
(шаг 2 — папка, шаг 3 — профиль/аргументы+Запуск). + просто по одному, − много кликов.

## M6 — Меню у кнопки «＋ Новая»
```
 [＋ Новая ▾]
   ├ ▶ Claude · E:\Scripts\Castellyn      (повторить последнее)
   ├ ▶ codex · E:\Scripts\Docx…
   │  ── Проекты ──
   ├ E:\Scripts\Castellyn
   ├ E:\Scripts\Docx и pdf обработка
   └ ⚙ Настроить запуск…
```
1 клик = запуск из истории/проектов. + очень быстро для повторов, − полная форма прячется в «Настроить».

## M7 — Проект-центричный (папка первична)
```
 Проект:  [E:\Scripts\Castellyn ▾]
 Запустить здесь:  [◆ Claude]  [</> opencode]  [◎ codex]  [>_ shell]
```
Сначала папка, потом клик среды = запуск в ней. + идеально ложится на «space=проект», − профиль/аргументы вторичны.

## M8 — Табличная (строка на среду)
```
 Среда        Профиль   Аргументы              
 ◆ Claude     [cc1 ▾]   [--effort high ...]    [▶]
 </> opencode   —       [ ................. ]  [▶]
 ◎ codex        —       [ ................. ]  [▶]
 Папка для всех:  [E:\Scripts\Castellyn ▾]
```
Видно все среды разом, у каждой свой запуск. − плотно, много всего.

## M9 — Горизонтальный бар (всегда виден, БЕЗ поповера)
```
 [◆][</>][◎][>_]  │  [E:\Scripts\Castellyn ▾]  │  [cc1 ▾]  │  [▶ Запустить]  ⚙
```
Одна строка под шапкой, ничего не разворачивается. + нет прыжков/поповеров, − узко на малых окнах.

## M10 — Быстрый + Подробный
```
 [  ▶ Запустить Claude здесь  ]          ← дефолты (профиль + папка проекта)
    ⚙ Настроить запуск…                  ← полная форма по желанию
```
Одна большая кнопка на 90% случаев + «Настроить» для остального. + минимум трения, − «здесь» надо чётко определить (папка проекта/space).

---

## Рекомендация
Под твой workflow («агент в той же папке», space=проект) лучше всего ложатся **M7 (проект-центричный)**
или **M10 (быстрый+подробный)**, опционально с **M6-меню** для повторов. M9 — если хочешь совсем без
поповера. Можно комбинировать (напр. M7 сверху + M6-меню на «＋»).
